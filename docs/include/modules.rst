============================
:index:`Modules`
============================

OpenDeskLab is comprised of different modules that can be assembled following some basic guidelines.

Assembly Procedure
~~~~~~~~~~~~~~~~~~

The modules are assembled together using M6 connecting screws through the four holes that exist in every module.
